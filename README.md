"# Issue-Tracker" 
1:var পুরনোভাবে variable বানানোর উপায়, এটা আবার change এবং declare করা যায়.

2:let দিয়ে value change করা যায় কিন্তু আবার declare করা যায় না.

4:const একবার value দিলে আর change করা যায় না.

5:spread operator (  ) দিয়ে array বা object কে copy বা expand করা যায়.

6:map() → নতুন array তৈরি করে.

7:filter() → শর্ত অনুযায়ী কিছু data রেখে নতুন array দেয়.

8:forEach() → শুধু loop চালায়, কিছু return করে না.

9:arrow function হলো function লেখার ছোট উপায় যেখানে => ব্যবহার করা হয়.

10:template literals হলো backtick ( ) দিয়ে লেখা string যেখানে ${} দিয়ে variable বসানো যায়.
